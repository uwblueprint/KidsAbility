Note: this was done in AWS Cloud9 on Linux	

	1. Create file
		- touch mongodb-org-3.6.repo
	2. Add this to file content:
		[mongodb-org-3.6]
		name=MongoDB Repository
		baseurl=https://repo.mongodb.org/yum/amazon/2013.03/mongodb-org/3.6/x86_64/
		gpgcheck=1
		enabled=1
		gpgkey=https://www.mongodb.org/static/pgp/server-3.6.asc
	3. Move file
		- sudo mv mongodb-org-3.6.repo /etc/yum.repos.d
	4. Install Mongo
		- sudo yum install -y mongodb-org
	5. Go to root directory and enter the following:
		- mkdir data
		- echo 'mongod --bind_ip=$IP --dbpath=data --nojournal'  > mongod
		- chmod a+x mongod
	6. Install mongoose in project
		- npm install --save mongoose
